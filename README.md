# AutoDealer Pro

Modern automotive dealership management dashboard.

## Getting Started

```sh
# Install dependencies
npm install

# Start the dev server
npm run dev

# Lint
npm run lint

# Build for production
npm run build
```

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deploy

Deploy the built `dist/` folder to any static hosting (GitHub Pages, Netlify, Vercel, etc.).
